# Footer

An Kidsagree footer component.

## Usage

```jsx
import { Footer } from '@kidsagree/ui'

const App = () => (
  <Footer />
)
```

## Properties

### `compact`

- Type: `Boolean`
- Default value: `false`

Set this property to `true` to display the footer without the menus.

#### Example:

```jsx
<Footer compact />
```
